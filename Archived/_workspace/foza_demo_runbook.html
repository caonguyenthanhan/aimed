FOZA Demo Runbook

Goal: run Agent mode with FOZA safely (no tool-calling timeouts), while keeping Graph evidence grounding.

1) Start CPU Server + Graph

  • Open Docker Desktop.
  • Run cpu_server/launcher/run_menu.bat (Admin) → option 1.
  • Or PowerShell: powershell -NoProfile -ExecutionPolicy Bypass -File "cpu_server\launcher\run.ps1" -Port 8000
  • Ensure CPU_SERVER_URL points to your reachable URL (local or ngrok).

2) FOZA Recommended Env

AGENT_PROVIDER
foza
AGENT_FORCE_ACTIONS
0 (Simple Mode, no tool calling)
FOZA_BASE_URL
https://api.foza.ai/v1
FOZA_TOKEN
(your token)
LLM_MODEL_NAME
(your FOZA model)

Simple Mode still includes system persona + conversation history + GRAPH_EVIDENCE.

3) Circuit Breaker (Recommended)

FOZA_CIRCUIT_FAIL_THRESHOLD
3
FOZA_CIRCUIT_OPEN_MS
600000 (10 min)

If FOZA times out repeatedly, the route skips FOZA temporarily instead of cascading into confusing downstream provider errors.

4) DB SSL Warning (Optional)

If you see a warning about sslmode=require/prefer/verify-ca, set your DB URL to use sslmode=verify-full explicitly.
