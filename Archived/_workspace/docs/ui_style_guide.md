# UI Style Guide (Project)

## Principles
- Consistency over novelty.
- Mobile-first layouts.
- Clear states: loading, empty, error.

## Stack
- Tailwind CSS + shadcn/ui components.

## Typography
- Prefer short sentences.
- Avoid walls of text; use small sections.

## Color
- Use semantic tokens (`primary`, `secondary`, `destructive`, etc.).
- Keep contrast accessible.
