# Thư mục tài liệu viết luận

Mục tiêu của `thesis/` là gom và “gắn nhãn” các tài liệu quan trọng trong repo để bạn viết luận nhanh, không bị rối bởi tài liệu kỹ thuật rải rác.

## Bắt đầu nhanh

- Dàn ý chương: xem [THESIS_OUTLINE.md](./THESIS_OUTLINE.md)
- Mapping chương ↔ tài liệu có sẵn: xem [CHAPTER_MAPPING.md](./CHAPTER_MAPPING.md)
- Danh mục nguồn nội dung (bạn có thể trích/đưa vào phụ lục): xem [CONTENT_SOURCES.md](./CONTENT_SOURCES.md)
- File tham khảo BibTeX (khung): xem [REFERENCES.bib](./REFERENCES.bib)

## Nguyên tắc sử dụng

- Ưu tiên trích nội dung từ các tài liệu “SSOT/chuẩn” trong repo (ARCHITECTURE, docs/ Memory Bank, báo cáo test).
- Khi cần copy sang luận văn, copy đoạn + ghi rõ nguồn (tên file và mục).
- Hình/flowchart: ưu tiên dùng file Mermaid hiện có và xuất ảnh (PNG/SVG) để chèn vào luận văn.
